# 照心录 / Zhaoxinlu Miniapp

Zhaoxinlu is a mini app for daily self-reflection. Users record small acts of virtue and moments of fault, seal them into a daily card, and review their inner balance over time.

## 当前方向

照心录先从“每日自我记录”开始：用户写下一条功或过，将纸符收入对应容器，并在当天结束时封存成一张今日卡片。

产品名称不绑定某一种具体物件。后续视觉载体可以继续探索为双匣、签筒、手札、账簿、灯盏或其他更契合用户偏好的形式。

## Prototypes

- `prototypes/boxes-demo/index.html`: 当前主要探索方向，包含双匣、纸符、封存卡片与归档查看。
- `prototypes/beans-demo/index.html`: 早期探索稿，包含赤豆/青豆、心灯与功过记录玩法。
