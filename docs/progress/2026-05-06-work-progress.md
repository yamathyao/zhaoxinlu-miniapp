## 已完成

- 新建并继续使用 `feat/configurable-default-resources` 分支开发。

- 默认主题资源已落到 `miniprogram/assets/themes/default/`，包括背景纸纹、匣子、卡片、印章、心灯等资源文件。
- 首页与封存卡片开始通过主题资源读取背景、匣、心灯等视觉素材。
- 将心灯抽成主题资源槽 `theme.assets.heartLamp`，并保留 CSS 兜底样式。
- 首页心灯状态已按功过数量差做阶梯变化：`1 / 3 / 5 / 10`，功多变亮，过多变暗，持平为平衡态。

- 今日页符纸区域改为根据屏幕高度动态填充到按钮上方。
- 匣中符文改为固定高度内堆叠展示，避免多个符文把匣子撑高。
- 符文飞入匣子动画改为和匣中静止符文更一致的样式。
- 修复 `paper-slip` 弹层打开态丢失导致“写一功”只显示背景、无符纸、无法回退的问题。
